CHIP-8 Interpreter

Introduction

A CHIP-8 virtual machine. It runs CHIP-8 programs, not SUPER-CHIP or XO-CHIP programs. Sound is currently not implemented.

Controls

The original computers with CHIP-8 interpreters had a keyboard that looked like this:

123C
456D
789E
A0BF

The default setting for this interpreter is a Qwerty keyboard which maps it to the following keys:

1234
QWER
ASDF
ZXCV

The corresponding buttons are used for Qwertz and Azerty layouts (select it from the menu).

Space pauses/unpauses the program.

CPU

There are a few instructions which have been implemented differently in different interpreters. This mostly concerns the shift instructions (8xy6 and 8xye), the read from/write to memory instructions (fx55 and fx65) and the jump with offset instruction (bnnn). Some program may require one or the other implementation of these instructions to work. You can choose them in the CPU menu.

Graphics

Note that the sprite flickering is not a bug but accurate behavior.

Debugging

The interpreter comes with some debugging functionalities.

  • Print instructions (hotkey p): prints all executed instructions to terminal while running the interpreter normally.
  • Step (hotkey n): only works when paused. Executes one instruction and prints it to terminal.
  • Show registers and memory (hotkey j): only works when paused. Opens a window with all registers and memory. It is only updated when using step (with or without printing).
  • Print registers and memory (hotkey k): only works when paused. Prints all registers and memory to terminal.
  • Step and print registers and memory (hotkey m): only works when paused. Combines the previous two.
